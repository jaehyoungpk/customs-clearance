# 통합 HS 코드 추천 시스템

한국 관세청 HS 코드를 지능적으로 추천하는 시스템입니다.

## 주요 기능

- **HS 원본 데이터 우선**: 공식 관세청 데이터를 최우선으로 활용
- **표준품명 통합**: 표준품명 데이터와 HS 코드 데이터 완전 통합
- **동의어 확장**: "나사" 검색 시 "볼트", "스크류" 등도 함께 검색
- **한국어 특화**: 한국어에 최적화된 의미 임베딩 모델 사용
- **지능적 필터링**: 관련 없는 검색 결과 자동 제거
- **LLM 분석**: OpenAI GPT를 활용한 정확한 분석 (선택사항)



## 사용 예시

```
🔍 상품명을 입력하세요: LED 조명
재질 (선택사항): 
용도 (선택사항):

   품목명: 발광다이오드 모듈
   신뢰도: 0.834
```

## 트러블슈팅

### 모듈 import 오류
```bash
pip install --upgrade sentence-transformers torch
```

### 메모리 부족 오류
- 배치 크기를 줄이거나 더 작은 모델 사용
- 가상 메모리 설정 확인

### 데이터 파일 오류
- 파일명과 경로 확인
- 파일 인코딩이 UTF-8인지 확인
- Excel 파일이 손상되지 않았는지 확인
